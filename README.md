
# Disgram

A simple and fast way to bridge a discord channel to a telegram chat.


## Authors

- [@dhanushlnaik](https://www.github.com/dhanushlnaik)


## Installation


```bash
  git clone https://github.com/dhanushlnaik/disgram.git
  cd disgram
```
    
## Run Locally

Clone the project

```bash
  git clone https://github.com/dhanushlnaik/disgram.git
```

Go to the project directory

```bash
  cd disgram
```

Install dependencies

```bash
  npm install --force
```

Rename the .env.example file to .env and fill all the required info/tokens.

```js
DISCORD_BOT_TOKEN=
TELEGRAM_BOT_TOKEN=
MONGO_URL=
DISCORD_WEBHOOK_ID=
DISCORD_WEBHOOK_TOKEN=
TELEGRAM_CHAT_ID=
DISCORD_CHANNEL_ID=
```
Start the server

```bash
  npm run start
```
